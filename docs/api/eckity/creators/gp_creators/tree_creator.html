110        Recursively add children to a function node.
111
112        Parameters
113        ----------
114        tree : List[TreeNode]
115            Total representation of the tree.
116        fn_node : FunctionNode
117            Function node, parent of the children to be created.

131                depth=depth + 1,
132                node_type=func_types[i],
133            )
